🗺️

.map() Method

Transforms

Creates a new array by calling a function on every element

students.map(student => student.name)
🔍

.filter() Method

Filters

Creates a new array with elements that pass a test

students.filter(s => s.marks > 60)
📊

.reduce() Method

Accumulates

Reduces array to a single value (left to right)

students.reduce((sum, s) => sum + s.marks, 0)
🔄

.forEach() Method

Iterates

Executes a function on each array element

students.forEach(s => console.log(s))

📝 Multiple Approaches:

Method 1: reduce() students.reduce((max, s) => s.marks > max.marks ? s : max)
Method 2: sort() [...students].sort((a,b) => b.marks - a.marks)[0]
Method 3: for loop let topper = students[0]; for(let s of students) if(s.marks > topper.marks) topper = s

📚 Array Methods Cheat Sheet

.map() Returns NEW array | Same length | Transforms each element
.filter() Returns NEW array | Shorter length | Keeps elements passing test
.reduce() Returns SINGLE value | Accumulates | Start with initial value
.forEach() Returns UNDEFINED | Side effects | Just loops through
